## 简介
FTPReader是一个适用于服务器文件自动传输的**免费**且**开源**的  
项目，可用于文件定时自动传输需求的用户。
## 目录
- 安全性
- 安装
- 使用
- 应用程序接口(API)
- 许可证
- 贡献者

## 安全性
 - 本软件**并不能**100%保证数据传输时文件的安全，因此，对于一些敏感文件传输，请不要使用本软件(重要！)
